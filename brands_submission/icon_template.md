## Design Tools

### Free Online Tools:
1. **Canva** - https://www.canva.com
   - Search for "lightning icon"
   - Customize colors
   - Export as PNG 256x256

2. **Figma** - https://www.figma.com
   - Professional design tool
   - Free for personal use
   - Export at multiple sizes

3. **Photopea** - https://www.photopea.com
   - Free Photoshop alternative
   - Works in browser
   - PNG export

4. **Icon Kitchen** - https://icon.kitchen
   - Generate app icons
   - Multiple sizes
   - Material Design style

### Paid Options:
- **Fiverr** - $5-20 for simple icon

## Export Settings

### icon.png
- Size: 256x256 pixels
- Format: PNG
- Background: Transparent
- Color mode: RGB

### logo.png (optional)
- Size: 512x256 pixels
- Format: PNG
- Background: Transparent
- Color mode: RGB

## Simple DIY Method

### Using Canva:
1. Go to https://www.canva.com
2. Create new design (256x256px)
3. Search "lightning bolt icon"
4. Customize color to #0072CE
5. Add subtle glow effect
6. Download as PNG (transparent background)

### Using Material Design Icons:
1. Go to https://pictogrammers.com/library/mdi/
2. Search "flash" or "lightning-bolt"
3. Download SVG
4. Convert to PNG at 256x256
5. Change color to Estonian blue

## When You Have Your Icon

1. Save as `icon.png` (256x256px)
2. Save as `logo.png` (512x256px) if you have one
3. Place in the `icons/` folder
4. Test in Home Assistant
5. Submit to brands repository when ready
