# Integration Icons

This folder contains placeholder icon files. Replace these with actual PNG images.

## Required Images

1. **icon.png** (256x256px)
   - Square icon with transparency
   - Used in integration tiles and cards
   - Should represent electricity/energy with Estonian theme

2. **icon@2x.png** (512x512px) - Optional
   - High-resolution version of icon
   - For retina displays

## Creating Icons

### Quick Options:
1. Use Material Design Icon generator
2. Create in Canva (free)
3. Use Figma
4. Hire designer on Fiverr
